# Examples

These are small pattern bundles copied from the repo's sample projects.

They are here so the skill can point to self-contained example files without depending on the live sample-project directories.

## Included Bundles

### `flash-fiction-loop/`
Minimal loop-through-a-list pattern:
- list exposed through variables
- prompt selects from `count`
- one output file per iteration

### `chapter-loop/`
Rolling long-form fiction pattern:
- chapter planning
- drafting from selective dossier context
- repetition auditing
- final chapter assembly
- summary generation

### `dossier-cascade/`
Section-by-section dossier build pattern:
- one reusable section-builder prompt
- canon-lock rules
- append-based cascade workflow

### `ping-smoke-test/`
Minimal manifest-and-payload sanity-check pattern:
- one variable
- one prompt block
- one pipeline
- one generated file

These bundles are examples, not canonical templates. Adapt them to the user's workflow.
